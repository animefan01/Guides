Guide to install java in wsl debian/ubuntu


Its very simple, only you need to do is download from internet using CURL, and build with dpkg.
open

sudo apt update
sudo apt install curl

Then you to openJDK website and copy the URL from dep file download

curl -o https://download.oracle.com/java/21/latest/jdk-21_linux-x64_bin.deb
dpkg -i jdk-21_linux-x64_bin.deb
